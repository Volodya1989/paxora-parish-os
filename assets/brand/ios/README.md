# iOS Brand Pack Exports (Source Input for IOS-A4)

Place approved brand-pack PNG exports in this folder before running:

```bash
npm run mobile:ios:assets
```

Required files:
- `icon-notification-20@2x.png` (40x40)
- `icon-notification-20@3x.png` (60x60)
- `icon-settings-29@2x.png` (58x58)
- `icon-settings-29@3x.png` (87x87)
- `icon-spotlight-40@2x.png` (80x80)
- `icon-spotlight-40@3x.png` (120x120)
- `icon-app-60@2x.png` (120x120)
- `icon-app-60@3x.png` (180x180)
- `icon-app-store-1024.png` (1024x1024)
- `splash-logo-1x.png` (120x120)
- `splash-logo-2x.png` (240x240)
- `splash-logo-3x.png` (360x360)

Optional dark-mode splash logos:
- `splash-logo-dark-1x.png` (120x120)
- `splash-logo-dark-2x.png` (240x240)
- `splash-logo-dark-3x.png` (360x360)

If optional dark files are omitted, IOS-A4 uses a single-style splash image for both appearances.
